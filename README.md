This is a Vue.js dashboard starter setup that includes the following libraries and tools:

- Vue
- Naive UI
- Tailwind
- Pinia
- Vue Router
- Eslint enabled
- Prettier enabled

To get started:
```sh
git clone https://github.com/iyoub/VNT-dashboard-starter.git

cd VNT-dashboard-starter

npm install

npm run dev
```
